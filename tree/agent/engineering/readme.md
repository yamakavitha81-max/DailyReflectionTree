# Daily Reflection Tree Assignment

## Overview
This project is a structured reflection system designed to help users think deeply about their daily work experiences.

## Part A: Reflection Tree
A deterministic decision tree that guides users through a series of connected, open-ended questions.

## Features
- No multiple-choice questions
- No judgment or scoring
- Encourages deep thinking
- Questions are interconnected

## Reflection Axes Covered
- Locus (internal vs external control)
- Orientation (past vs future thinking)
- Valence (positive vs negative experiences)

## Part B: AI Agent (Optional)
An AI assistant that:
- Identifies tone (positive/negative/neutral)
- Identifies control (internal/external)
- Selects the next question from the reflection tree

## Guardrails
- No hallucinated responses
- AI cannot generate new questions
- Only predefined structure is followed

## How It Works
User answers → system analyzes → next question is selected → deeper reflection continues

## Conclusion
This system promotes structured, meaningful, and unbiased reflection.
